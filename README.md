# YT-Downloader
A simple Python GUI app for downloading YouTube videos or audio using yt-dlp and FFmpeg.

## Features
- Download videos in **Full HD (MP4)**  
- Download **audio or video only (MP3, MP4)**  
- Support for **multiple links at once**  
- Choose **output folder**  
- Clean and intuitive GUI
